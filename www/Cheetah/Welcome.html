Welcome to Cheetah

 
 

Cheetah is a set of programs for processing data obtained at free electron laser sources, particularly data obtained in a serial manner. 

 

Acknowledgements

The development of Cheetah has been led by Anton Barty at the Center for Free-Electron Laser Science at the Deutsches Elektronen-Synchrotron, DESY in Hamburg, Germany.
Code has been contributed by: Filipe Maia, Rick Kirian Thomas White, Chuck Yoon, Andrew Aquila.  Code has been tested on many beamtimes by Nadia Zatsepin and others.
